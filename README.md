# generator-mycli [![NPM version][npm-image]][npm-url]
> cli generator

## 安装

```bash
npm init yo mycli
```

## 安装速度太慢解决

由于某些原因，建议安装nrm来修改npm的镜像，以提高npm下载速度

```bash
npm i -g nrm
```

设置为淘宝镜像

```bash
nrm use cnpm
```

查看当前所有镜像，带*标识的为当前所使用的镜像

```bash
nrm ls
```
